Barbershop Borodinski
=====================

Собрать проект:
```bash
make
```

Пересобрать проект целиком:
```bash
make -B
```

Пересобрать только результаты работы enb:
```bash
make enb-make -B
```

Запустить watch (проект будет доступен по адресу [localhost:8080](http://localhost:8080)):
```bash
make watch
```

Проверить код-стайл:
```bash
make test
```
